Зураг 6. Өгөгдлийн ерөнхий схем

ӨЕС өргөтгөл

Хүснэгт 1. Хэрэглэгч (User)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id UserID PK integer Хэрэглэгчийн дахин давтагдашгүй дугаар
нэр UserName varchar(100) Хэрэглэгчийн бүртгэлтэй нэр
имэйл Email varchar(150) Нэвтрэхэд ашиглах имэйл хаяг
нууц_үг Password text Нэвтрэх нууц үг (hash)
утас Phone varchar(20) Хэрэглэгчийн утасны дугаар
эрх Role varchar(10) user / admin төрлийн эрх
Хүснэгт 2. Хаяг (Address)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id AddressID PK integer Хаягийн дахин давтагдашгүй дугаар
хэрэглэгч_id UserID FK integer Аль хэрэглэгчийн хаяг болохыг илэрхийлнэ
хаяг Address text Дэлгэрэнгүй хаяг
дүүрэг District varchar(100) Хамаарах дүүрэг
Хүснэгт 3. Захиалга (Order)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id OrderID PK integer Захиалгын дахин давтагдашгүй дугаар
хэрэглэгч_id UserID FK integer Захиалга өгсөн хэрэглэгч
нийт_үнэ TotalPrice decimal(10,2) Захиалгын нийт төлбөр
хаяг DeliveryAddress text Хүргэлтийн хаяг
төлөв Status varchar(50) Хүлээгдэж буй / Хүргэгдсэн / Цуцлагдсан
огноо CreatedAt timestamp Захиалга өгсөн огноо цаг
Хүснэгт 4. Сагс (Cart)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id CartID PK integer Сагсны бичлэгийн дахин давтагдашгүй дугаар
хэрэглэгч_id UserID FK integer Сагсанд нэмсэн хэрэглэгч
меню_хоол_id MenuItemID FK integer Сагсанд нэмэгдсэн меню хоол
тоо_ширхэг Quantity integer Захиалсан тоо ширхэг
нэмэлт_орц Extras json Хэрэглэгчийн сонгосон нэмэлт орцууд
Хүснэгт 5. Ангилал (Category)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id CategoryID PK integer Ангиллын дахин давтагдашгүй дугаар
нэр CategoryName varchar(100) Ангиллын нэр (Бургер, Пицца гэх мэт)
Хүснэгт 6. Меню хоол (Menu Item)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id MenuItemID PK integer Меню хоолны дахин давтагдашгүй дугаар
ангилал_id CategoryID FK integer Хамаарах ангилал
нэр ItemName varchar(200) Хоолны нэр
үндсэн_үнэ BasePrice decimal(10,2) Үндсэн үнэ (төгрөгөөр)
үндсэн_калори BaseCalories integer Үндсэн калорийн тоо (kcal)
тайлбар Description text Хоолны дэлгэрэнгүй тайлбар
Хүснэгт 7. Орц (Ingredient)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id IngredientID PK integer Орцны дахин давтагдашгүй дугаар
меню_хоол_id MenuItemID FK integer Аль меню хоолны орц болохыг илэрхийлнэ
нэр IngredientName varchar(100) Орцны нэр
нэмэлт_үнэ ExtraPrice decimal(10,2) Орц нэмэхэд нэмэгдэх үнэ
Хүснэгт 8. Захиалгын түүх (History)
Баганын нэр ӨС нэр Түлхүүр Төрөл Тайлбар
id HistoryID PK integer Түүхийн бичлэгийн дахин давтагдашгүй дугаар
хэрэглэгч_id UserID FK integer Үйлдэл хийсэн хэрэглэгч
захиалга_id OrderID FK integer Ямар захиалгатай холбоотой үйлдэл
үйлдэл Action varchar(100) Захиалга өгөх, цуцлах, хүлээн авах үйлдэл
огноо CreatedAt timestamp Үйлдлийн огноо цаг
